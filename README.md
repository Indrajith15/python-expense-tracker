# Python Expense Tracker

A simple command-line Expense Tracker built using Python.

## Features

- Add new expenses
- Record the date of each expense
- Categorize expenses
- View all recorded expenses
- Calculate total expenses
- Save expenses using JSON
- Automatically load saved expenses

## Technologies Used

- Python
- JSON
- File Handling


## How to Run

python expense_tracker.py

## Example Menu

******** EXPENSE TRACKER ********

1. Add Expense
2. View Expenses
3. Show Total Expense
4. Exit
